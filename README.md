  - C Compiler: Armcc `Ver 5.06 update 7 (build 960)`
  - Assembler: Armasm `Ver 5.06 update 7 (build 960)`
  - Linker/Locator: ArmLink `Ver 5.06 update 7 (build 960)`
- Debugger: J-Link `Ver 9.40`
